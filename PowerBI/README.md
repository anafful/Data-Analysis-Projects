# Power BI Labs & Exercises

## Overview

This section contains hands-on Power BI exercises focused on building foundational skills in data loading, transformation, and report design.

The work emphasises practical familiarity with Power BI workflows rather than full production dashboards.

---

## Areas Practised

- Data import and connectivity  
- Data cleaning and transformation  
- Basic data modelling concepts  
- Report and visual design  
- Interactive filtering and layout  

---

## Key Skills Developed

✔ Working with datasets inside Power BI  
✔ Transforming data for analysis  
✔ Creating structured reports  
✔ Understanding visual best practices  

---

## Notes

These exercises were completed as part of guided learning and skill development.

The focus was on understanding Power BI mechanics, interface navigation, and core analytical features.
